# Password Manager Project (Java)

## Description
A simple console-based Password Manager application developed using Java.
This project allows users to store and retrieve passwords securely using file handling.

## Features
- Master password protection
- Add new website credentials
- View saved passwords
- File-based storage

## Technologies Used
- Java
- File Handling
- OOP Concepts

## How to Run
1. Compile the program:
   javac PasswordManager.java
2. Run the program:
   java PasswordManager
